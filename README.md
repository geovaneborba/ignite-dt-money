## :dart: Sobre

<p>
O Dt Money foi projeto construido em aula durante o módulo de <strong>Consumo de API e performance no ReactJS</strong> que faz parte do programa de especialização Ignite da <a href='https://www.rocketseat.com.br/' target="_blank">Rocketseat</a>.

Neste projeto foi construido uma aplicação web, conectando a uma API fake (json-server) para persistir os dados em um arquivo json. Além de formas de melhorar a performance da aplicação e o entendimento do fluxo de renderização.

</p>

## :checkered_flag: Começando

```bash
# Clone este repositório
$ git clone https://github.com/geovaneborba/ignite-dt-money.git

# Entre na pasta
$ cd ignite-dt-money

# Instale as dependências
$ npm install

# Inicie API Fake (json-server)
$ npm run dev:server

# Para iniciar o projeto
$ npm run dev
# O app vai inicializar e será informado no terminal o endereço e porta onde estará rodando a aplicação
# geralmente é http://localhost:5173/
```
